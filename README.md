# Optimizador de Envíos

## Descripción
Este repositorio contiene la documentación funcional del proyecto **Optimizador de Envíos**, enfocada en la definición del producto, las historias de usuario y la descomposición de trabajo en subtareas.

## Objetivo

Definir de forma clara el alcance, las reglas y las necesidades funcionales de un sistema que permita recomendar opciones de envío según criterios de costo y tiempo de entrega.

## Contenido del repositorio

- `PRD.md`: documento de definición del producto, alcance, reglas de negocio y requisitos generales.
- `USER_STORIES.md`: historias de usuario del sistema con criterios de aceptación, DoR y DoD.
- `SUBTASKS.md`: desglose de trabajo en subtareas para la implementación de las historias de usuario.

## Alcance general

- El sistema aplica únicamente para envíos dentro de Colombia.
- El cálculo de recomendaciones considera origen, destino, peso y prioridad del usuario.
- Las prioridades disponibles son menor costo y menor tiempo de entrega.
- El sistema debe mostrar una recomendación principal y, cuando existan, opciones alternativas.
